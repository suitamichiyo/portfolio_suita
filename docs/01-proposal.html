1 プロジェクト概要

サービス名 必須
制作するWebサービスの名前

inimu集客増を目指した来店理由形成型ホームページ再編

コンセプト 必須
サービスを一言で表す説明（キャッチコピー＋1〜2文の説明）

浅草で出会う、私だけの香り。

30代女性・感度層の「失敗したくない香り選び」課題を、「浅草」×「香り体験」×「日本のものづくりとの出会い」で解決する。来店前の接点づくりから店頭体験、ECでの再購入までを自社HP導線でつなぎ、来店理由と継続購入の両方を育てる。

特別なお土産も、香り体験も。浅草で“選ぶ理由”をつくる。

ターゲットユーザー 必須
このサービスを使う主なユーザー像（概要レベル）

パターン1 都内勤務の感度層女性
30〜34歳 / 都内企業勤務の会社員
香りは好きだが、自分に合うものを失敗せず選びたい。量産的な商品ではなく、背景やストーリーのある“少し特別な香り”に出会いたい。浅草らしさや日本のものづくりを感じられる体験にも魅力を感じる。
主な利用シーン: 週末の浅草おでかけ、自分用の香り探し、カフェ・雑貨店巡りの延長での来店

パターン2 ギフト・土産目的の感度層女性
32〜38歳 / 会社員 / フリーランス / 販売・企画職
友人やパートナーに贈るとき、ありきたりではない“理由のあるギフト”を選びたい。見た目だけでなく、香り・背景・浅草で買う意味まで含めて納得して選びたい。自分用にも欲しくなるような商品や体験を比較しながら探したい。
主な利用シーン: 誕生日・記念日のギフト探し、浅草土産探し、旅行やおでかけのついでの立ち寄り

パターン3 体験重視の2名来店層
28〜35歳 / 会社員 / カップルの片方 / 友人同士で行動する社会人
せっかく浅草に行くなら、買うだけでなく“体験した記憶”も持ち帰りたい。写真映えだけでなく、自分たちらしい特別感や思い出になる体験を探している。体験後に商品購入や再購入にもつながる場所だと魅力を感じる。
主な利用シーン: 週末デート、友人とのおでかけ、記念日・旅行中の体験利用

2 背景と課題

市場の現状 必須
このサービスが存在する市場の現状と課題

国内フレグランス需要は拡大基調にあり、2027年には674億円規模となる予測もある。ニッチフレグランスやメゾン系、高価格帯フレグランスの受容も進んでおり、価格だけではなく世界観や背景を含めて選ぶ傾向が強まっている。

浅草周辺でも、調香体験型・香水専門型・観光土産型の競合が複数存在し、来店前の比較検討先は増えている。加えて、訪日客増加と浅草観光需要の高まりにより、「浅草でどこに行くか」を事前に選ばれる構造への対応が必要になっている。

3C分析（Customer） 必須
ターゲット顧客の特徴・ニーズ・行動・市場トレンドを4観点で整理

1. ターゲット顧客の特徴（年齢層・職業・ライフスタイル）
年齢層は20〜40代が中心で、自社コア年代は30代女性。日本のものづくり・文化・人の交流に興味があり、新しい体験にアンテナを張っている層で、大量生産より良質な製品を好み、嗜好品・雑貨・体験との出会いを大切にする。国内外の観光客は、地域に特化した特別なお土産や記憶に残るものを求めており、背景や物語のある消費を好むライフスタイルが特徴である。

2. 顧客のニーズや悩み
香りを購入する際に最も重視されるのは「香り（品質）」そのものであり、好印象を与えること、リラックス、気分転換が期待されている。使用シーンは出勤前・人に会う前、仕事の切り替え、外出・会食前、帰宅後のリラックス、休日のお出かけ・旅行・ギフトまで幅広い。一方で「自分に合う香りが分からない」「強すぎる香りが苦手で失敗したくない」「人と被りたくない」「買う根拠・語れる理由が欲しい」といった悩みがあり、選定時には口コミや他人の意見を参考にする層が過半数を占める。

3. 顧客の行動パターン（情報収集・購買行動）
情報収集では口コミ・レビュー参照が中心で、SNS、特にInstagramでの視覚的な情報収集も活発である。購入場所は百貨店や通販（Amazon・楽天）が主要な選択肢であり、体験型の代替先として ann fragrance（浅草）や NOSE SHOP（都市部）等が比較対象になる。低価格代替としてはドラッグストアの香り商品、ボディミスト、柔軟剤が使われる。香りの持続時間は3〜5時間を想定する層が中心で、「特別な日・ギフト（誕生日・記念日・手土産）」用途も一定数存在する。

4. 市場のトレンド
国内フレグランス市場（化粧品カテゴリ）は2024年に前年比15%増の575億円、2025年に600億円超、2027年には674億円と予測されている。関心層の増加を背景にメゾン・ニッチ系フレグランスが好調で、高価格帯（単価1万円以上）へのシフトも鮮明である。機能性と希少性が購買を牽引し、インバウンド需要の取り込みも活発化しているため、「日本らしさ」「浅草でしか買えない」という訴求が有効である。加えて、「背景が語れる香り」「日本素材」「トレーサビリティ」への関心が高まっている。

3C分析（Competitor） 必須
浅草周辺の主要競合3店を、特徴・強み・弱み・差別化ポイントで比較

1. ann fragrance 浅草アトリエ
サービス名: ann fragrance 浅草アトリエ。
特徴: 5種までの香りブレンド ¥6,050 / 単品香り ¥5,500（いずれも50ml・オリジナルラベル付き）、独特の世界観インテリアでエレガントな空間設計、調香師を「エヴァリュエーター」と呼ぶ独自表現、Googleマップ上で複数予約導線を確保している。
強み: SNS映えする独自インテリア・世界観の完成度が高く、「エヴァリュエーター」という肩書きによるブランディング力と、浅草エリアで先行している体験型の認知度がある。
弱み: 日本文化背景・素材ストーリーの深掘りが弱く、「日本素材特化」「産地トレーサビリティ」という文化的深度がない。
inimuとの差別化ポイント: inimuは「日本素材×産地背景×19年OEM実績」という文化的深度で上回り、「浅草限定・語れる理由のある香り」という情緒価値で差別化できる。annが世界観重視のエレガント系であるのに対し、inimuは「日本のものづくりとの出会い」という体験価値軸を打ち出せる。

2. ふれぐらんす工房「香」
サービス名: ふれぐらんす工房「香」 フレグランス 手作り体験 東京・浅草。
特徴: 2025年11月17日新規オープン、30分 ¥7,700 / 90分 ¥9,800（いずれも50ml）、体験の分かりやすさを重視した店舗設計、GoogleマップにHP以外の予約サイトも掲載。
強み: 体験内容の分かりやすさ・入りやすさを重視した設計で、新規オープン直後にもかかわらず口コミ件数・評価が急速に蓄積中（420件・5.0pt）。
弱み: 2025年11月開業のため実績・ブランド信頼の蓄積がまだ浅く、日本素材特化・産地ストーリーという独自軸がない。inimuと比較して料金も割高（30分¥7,700〜）である。
inimuとの差別化ポイント: inimuは2008年創業19年のOEM実績と協業先実績を信頼の証明として可視化でき、¥5,500という競合比で割安な料金設定に加え「体験後に持ち帰るもの・語れる背景」で納得感を補強できる。ブランドの歴史的深度と日本素材への特化で、新規参入競合との明確な差を打ち出せる。

3. よろし化粧堂 仲見世店
サービス名: よろし化粧堂 仲見世店。
特徴: 仲見世通り内の土産店で、¥300〜¥2,000の大正ロマンデザインのオリジナルスキンケア・香りものを展開。体験型ではなく物販型で、衝動買い・観光土産に特化した設計。
強み: 仲見世という国内最高クラスの人流立地、手軽な価格帯（¥300〜）で観光客の衝動買い需要に最適化されており、大正ロマンという分かりやすいデザイン訴求で土産用途に強い。
弱み: 体験型コンテンツがなく「作る・試す・深く探究する」という体験価値がない。価格帯が低く「特別な一本・語れる香り」という情緒価値が薄く、香りへの深い関与を促す仕掛けもない。
inimuとの差別化ポイント: 衝動買い型の土産需要に対し、inimuは「意味を選ぶ体験」として高関与購買に昇格できる。「大正ロマン×お土産」vs「日本素材×日本のものづくりとの出会い」で顧客層・価値軸が異なり、単価帯も大きく異なるため、観光客の中でも「体験・質・語れる背景」を重視する層をinimuが受け持てる。

解決すべき課題 必須
本プロジェクトで解決を目指す具体的な課題（3つ程度）

1. フレグランス需要拡大の中で、inimuならではの香り体験やブランド背景をWeb上で十分に伝え、比較検討時に選ばれる理由を明確にする必要がある。

2. 浅草周辺の競合が増える中で、調香体験・ギフト性・日本のものづくりとの接点といった独自価値を整理し、来店前の意思決定を後押しする導線設計が求められる。

3. 訪日客増加と観光需要の高まりを踏まえ、浅草来訪前の情報接触から店頭体験、EC再購入までを切れ目なくつなぎ、単発来店で終わらせない仕組みが必要である。

ユーザーの代替手段 必須
ユーザーが来店や予約前に比較・利用している代替手段

ワークショップ予約は現状ほぼじゃらん依存とみられ、予約導線の主導権が自社サイト外にある可能性が高い。

来店前の比較先は、近隣の調香体験店、香水専門店、観光土産店、Googleマップ、Instagram、一般ECに分散しており、ユーザーは複数チャネルを横断しながら意思決定している。

現行サイトはEC色が強く、来店目的別の比較検討がしづらいため、「体験したい」「ギフトを探したい」「浅草で特別な香りに出会いたい」といった来店動機に応じた選び方を支えきれていない。

ターゲットが抱える課題 必須
ターゲットユーザーが来店前に感じている具体的な悩み

1. 浅草で香り体験をしたい人にとって、近隣に複数の選択肢があり、どこが自分に合うか判断しづらい。

2. 香り商品は選択肢が多く、初心者ほど失敗回避ニーズが強い。

3. 体験・商品・土産・背景性を一度に満たせる店として、inimuの価値が事前に伝わりにくい。

現在の解決方法の不満点 必須
ユーザーが現在使っている代替手段に対する不満や不足

1. 近隣の体験型競合は予約しやすく内容も分かりやすい一方、土産・物販・背景ストーリーとの接続が弱い場合がある。

2. 観光土産店は買いやすいが、香り体験や作り手背景まで比較しにくい。

3. 一般ECや百貨店では買えても、浅草で行く理由や日本のものづくり文脈は伝わりにくい。

なぜ今この課題を解決する必要があるのか 必須
今このタイミングで取り組むべき理由

1. 浅草周辺の競合選択肢が増える中で、inimuが「何のために行く場所か」を事前に明確化できないと、比較の中で埋もれやすい。

2. 一方で、訪日需要増加と浅草観光需要の高まりは追い風であり、今なら来店理由形成型サイトへの再編効果が大きい。

3. 自社集客・自社導線を強化しないと、じゃらん依存から抜けづらい。

3 提案するソリューション

サービスの特長 必須
競合と差別化するサービスの特長を3つ

1. 来店目的別導線: 「特別なお土産を探す」「香りを試したい」「自分だけの香りを作りたい」「日本のものづくりに触れたい」で入口を分け、ユーザーが自分に合う体験を見つけやすくする。

2. 商品 / 体験の比較支援: 香りの系統、強さ、用途、ギフト向き、初心者向き、限定性に加え、「体験したい」「土産を買いたい」「背景のある香りに触れたい」といった目的別に選びやすく整理する。

3. 来店 / 予約 / 問い合わせ接続: 浅草店情報、体験概要、予約導線、FAQ、問い合わせを一貫導線でつなぎ、来店前の不安を減らしながら予約・来店へつなげる。

ユーザーが得られるメリット 必須
この提案によってユーザーが来店前後で得られる価値

1. 来店前に「ここへ行く理由」が明確になることで、比較検討の迷いが減る。

2. 「失敗しない香り選び」が体験を通じて解決され、来店中の満足度が高まる。

3. 体験・商品・土産の選択がつながり、来店後の購買まで納得感を持って進められる。

4 制作範囲

Webサイト（フロントエンド）の範囲 必須
制作するWebサイトのページ一覧と概要

メインページ（8ページ）

1. TOP｜トップページ
目的: 来店前に「行く理由」を3秒で形成する唯一のファーストビュー。3つの来店目的（お土産 / 体験 / 香水）への入口を一画面で伝える。
主要コンテンツ: 全画面ヒーロービジュアル、キャッチコピー「浅草で出会う、私だけの香り。」、来店目的別CTA×3、商品3層プレビュー、信頼実績バッジ、Instagram最新フィード、アクセス・営業日案内。

2. WORKSHOP｜ワークショップ体験ページ
目的: WS予約の主軸ページ。じゃらん依存から自社HP予約への移行を促す核。土日祝のみ運営を明記し事前計画予約を促す。
主要コンテンツ: 大型ビジュアル、体験の流れ4STEP図解、プラン・料金、FAQ、お客様の声、予約CTA、自社サイト推奨とじゃらん併記、SNS撮影スポット誘導。

3. PRODUCTS｜商品ページ
目的: 「失敗しない香り選び」ニーズに応える商品紹介ページ。3層商品構造を来店目的別に再編集し、EC購入・再購入導線として機能させる。
主要コンテンツ: WANOWA / PERFUMERS / 破天荒、破天荒特集、アロマレコルト、社会性商品、ギフトセット提案、各シリーズトップページへの導線。

4. STORY｜ブランドストーリーページ
目的: 「なぜinimuか」の根拠を伝え、30代女性感度層・インバウンドの共感と信頼を形成する。
主要コンテンツ: 日本素材クローズアップ写真、ブランドの歴史・想い、日本素材MAP、OEM協業実績、WANOWA植樹活動、代表プロフィール。

5. VISIT｜アクセス・店舗案内ページ
目的: 来店直前の最終接点。浅草観光中の動線と営業日を分かりやすく伝え、Googleマップ口コミ誘導で件数増加を仕組み化する。
主要コンテンツ: 店舗外観・内装写真、Googleマップ埋め込み、営業日カレンダー、営業時間・定休日・住所、浅草寺・駅からの徒歩ルート、Googleマップ口コミと投稿ボタン。

6. ONLINE｜オンラインショップページ
目的: WS体験後のEC再購入・遠方ファン向けの購買導線。体験からEC再購入への循環を設計する。
主要コンテンツ: EC商品一覧、カテゴリタブ絞り込み、「体験した香りをもう一度」セクション、破天荒の中央強調、ギフトラッピング・熨斗対応案内、LINE公式アカウント登録CTA。

7. RESERVE｜予約フォームページ
目的: じゃらん依存脱却の核機能。顧客データ取得・LINE連携・リピート設計の起点。
主要コンテンツ: カレンダー型空き枠表示、時間帯選択、プラン・人数選択、顧客情報入力フォーム、予約完了・サンクスページ、じゃらんリンク併記。

8. CONTACT｜お問い合わせページ
目的: 一般問い合わせ・法人・OEM相談の受け口。法人・ホテル向けBtoBリード獲得にも活用する。
主要コンテンツ: 問い合わせカテゴリ選択、入力フォーム、自動返信メール案内、法人・OEM実績紹介、送信完了ページ。

Webアプリケーション（サーバーサイド）の範囲 必須
制作するWebアプリケーションの機能概要

認証・セキュリティ
ログイン認証（メールアドレス + パスワード、Spring Security、未認証時は `/admin/login` へリダイレクト）と、管理者 / スタッフのロールベース権限管理を実装する。スタッフは予約・顧客・問い合わせに限定し、売上レポート・設定画面は管理者のみアクセス可能とする。優先度: Must。

予約管理
予約一覧・検索、予約詳細・編集、当日飛び込み対応の新規登録、キャンセル / 入場処理、月次 / 週次カレンダー表示、空き枠リアルタイム管理API、予約確認メール自動送信を実装する。日付・ステータス・氏名検索、CSV出力、スタッフメモ、時間帯別残枠、メールテンプレート編集に対応する。優先度: Must。

顧客管理
顧客一覧・検索、顧客詳細、顧客新規登録を実装する。氏名・メール・電話番号検索、LINE登録済み / リピーター / 初回フィルター、CSV出力、予約履歴、スタッフメモ表示に対応する。優先度: 一覧・検索 / 詳細はMust、新規登録はShould。

問い合わせ管理
問い合わせ一覧・ステータス管理、問い合わせ詳細・対応メモ、自動返信メール送信を実装する。カテゴリは一般 / WS / ギフト・法人 / OEM開発、ステータスは未対応 / 対応中 / 完了に対応し、自動返信には「3営業日（火〜日）以内」を明記する。優先度: Must。

商品管理
商品一覧・表示管理、商品情報編集を実装する。カテゴリ別表示、HP上の表示 / 非表示、看板商品フラグ、商品名・説明文・価格・素材情報・産地・画像アップロード編集に対応する。優先度: Should。

売上・レポート
売上レポートとWS実績サマリーを管理者向けに実装する。月次 / 年次 / カスタム集計、グラフ表示、CSV / PDF出力、予約件数、キャンセル率、曜日別分布、リードタイム別分布、じゃらん vs 自社予約比率の推移を可視化する。優先度: Should。

設定
予約カレンダー設定、WSプラン設定、ユーザー管理、メールテンプレート設定を実装する。営業日、時間帯、定員、臨時休業日、プラン名、料金、所要時間、公開 / 非公開、スタッフアカウント作成 / 編集 / 削除、権限設定、予約確認メール・問い合わせ自動返信テンプレート編集に対応する。優先度: カレンダー設定 / WSプラン設定 / ユーザー管理はMust、メールテンプレート設定はShould。

ダッシュボード
本日の予約件数、今月売上速報、未対応問い合わせ件数、直近7日間の予約カレンダープレビューを一画面で確認できるホーム画面を実装する。優先度: Must。

優先度サマリー
Must 16機能: 認証・予約管理・顧客管理・問い合わせ・カレンダー設定・ダッシュボード。Should 8機能: 売上レポート・商品管理・メールテンプレート。Could 0機能: 今回スコープ外。
