主要リレーション
- customers 1 : N reservations
- users 1 : N reservations
- customers 1 : N contacts
- workshop_plans 1 : N reservations
- calendar_settings 1 : N workshop_slots
- products 1 : N product_images
- email_templates 1 : N mail_logs

補足
- reservations と contacts は mail_logs と関連し、通知メール送信履歴を管理する
- products はシリーズ（破天荒 / PERFUMERS / WANOWA）単位と個別商品情報を管理する
- 多対多の主要要件は現時点で想定せず、基本は 1対多 構成とする

3 インデックス設計

インデックス一覧 必須
テーブル名インデックス名対象カラム種類設定理由
usersuk_users_emailemailUNIQUEログインIDとして一意性が必要なため
customersidx_customers_namenameINDEX氏名検索で頻繁に使用するため
customersidx_customers_emailemailINDEXメールアドレス検索に使用するため
customersidx_customers_phonephoneINDEX電話番号検索に使用するため
reservationsidx_reservations_customer_idcustomer_idINDEX外部キー結合と顧客別参照に使用するため
reservationsidx_reservations_user_iduser_idINDEX登録担当者別参照に使用するため
reservationsidx_reservations_plan_idworkshop_plan_idINDEX外部キー結合に使用するため
reservationsidx_reservations_date_statusreservation_date, statusINDEX日付・状態での一覧検索と絞り込みに使用するため
reservationsidx_reservations_date_timereservation_date, time_slotINDEX空き枠確認と時間帯別参照に使用するため
contactsidx_contacts_customer_idcustomer_idINDEX顧客との関連参照に使用するため
contactsidx_contacts_category_statuscategory, statusINDEX問い合わせ一覧の絞り込みに使用するため
workshop_plansidx_workshop_plans_activeis_activeINDEX有効プランのみ表示するため
calendar_settingsuk_calendar_settings_business_datebusiness_dateUNIQUE営業日を一意に管理するため
workshop_slotsidx_workshop_slots_calendar_idcalendar_setting_idINDEX営業日単位の時間帯取得に使用するため
workshop_slotsidx_workshop_slots_calendar_timecalendar_setting_id, slot_timeUNIQUE同一営業日に同一時間帯が重複しないようにするため
productsuk_products_slugslugUNIQUEURL識別子として一意性が必要なため
productsidx_products_series_visibleseries, is_visibleINDEXシリーズ別・公開状態別の絞り込みに使用するため
product_imagesidx_product_images_product_idproduct_idINDEX商品ごとの画像取得に使用するため
product_imagesidx_product_images_sort_orderproduct_id, sort_orderINDEX表示順で画像を並べるため
email_templatesuk_email_templates_nametemplate_nameUNIQUEテンプレート名の重複を防ぐため
mail_logsidx_mail_logs_template_idtemplate_idINDEXテンプレート別送信履歴参照に使用するため
mail_logsidx_mail_logs_reservation_idreservation_idINDEX予約関連メールの履歴参照に使用するため
mail_logsidx_mail_logs_contact_idcontact_idINDEX問い合わせ関連メールの履歴参照に使用するため
mail_logsidx_mail_logs_sent_atsent_atINDEX送信日時順の参照・監査に使用するため
