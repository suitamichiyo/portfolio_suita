## ページ別詳細

### 1. TOP｜トップページ

- **目的：** 来店前に「行く理由」を3秒で形成する唯一のファーストビュー。3つの来店目的（お土産／体験／香水）への入口を一画面で伝える
- **主要コンテンツ：**
  1. 全画面ヒーロービジュアル
  2. キャッチコピー「浅草で出会う、私だけの香り。」
  3. 来店目的別CTA×3（お土産を探す／体験を予約する／香水を選ぶ）
  4. 商品3層プレビュー（WANOWA／PERFUMERS／破天荒）
  5. 信頼実績バッジ（19年・4.8★・539件）
  6. Instagram最新フィード（@inimu_fragrance）
  7. アクセス・営業日案内

---

### 2. WORKSHOP｜ワークショップ体験ページ

- **目的：** WS予約の主軸ページ。じゃらん依存から自社HP予約への移行を促す核。土日祝のみ運営を明記し事前計画予約を促す
- **主要コンテンツ：**
  1. 大型ビジュアル（WS体験写真・完成ボトル）
  2. 体験の流れ（4STEP図解：選ぶ→ブレンド→詰める→持ち帰る）
  3. プラン・料金（¥5,500/名・土日祝のみ・11:00〜12:00 / 13:00〜14:00 / 15:00〜16:00）
  4. FAQ（初心者・当日予約・持ち物・キャンセル等）
  5. お客様の声（4.8pt・181件）
  6. 予約CTA（自社サイト推奨＋じゃらん併記）
  7. SNS撮影スポット誘導（#inimu_fragrance）

---

### 3. PRODUCTS｜商品ページ

- **目的：** 「失敗しない香り選び」ニーズに応える商品紹介ページ。3層商品構造を来店目的別に再編集。EC購入・再購入導線として機能させる
- **主要コンテンツ：**
  1. 3主力フレグランス（WANOWA／PERFUMERS／破天荒）
  2. 破天荒特集セクション（黒背景・独立ブロック）
  3. アロマレコルト（ライフスタイル）
  4. 社会性商品（COCKTAIL SAVON／JOY'N／MUSUBI）
  5. ギフトセット提案（¥5,000〜）
  6. 各シリーズトップページへの導線

---

### 4. STORY｜ブランドストーリーページ

- **目的：** 「なぜinimuか」の根拠を伝え、30代女性感度層・インバウンドの共感と信頼を形成する
- **主要コンテンツ：**
  1. 日本素材クローズアップ写真（ヒノキ・伽羅・ゆず等）
  2. ブランドの歴史・想い（KARA INNOVATE創業2008年・19年）
  3. 日本素材MAP（WANOWAの産地・特徴）
  4. OEM協業実績（ACTUS／青山フラワーマーケット／TRUNK HOTEL等）
  5. WANOWA植樹活動
  6. 代表プロフィール（清水社長・縦長写真・ナラティブ型）

---

### 5. VISIT｜アクセス・店舗案内ページ

- **目的：** 来店直前の最終接点。浅草観光中の動線と営業日を分かりやすく伝える。Googleマップ口コミ誘導で件数増加を仕組み化する
- **主要コンテンツ：**
  1. 店舗外観・内装の大型写真
  2. Googleマップ埋め込み＋ルート検索ボタン
  3. 営業日カレンダー（全日表示・土日祝のみ黒塗り）
  4. 営業時間・定休日・住所（火〜日・月曜祝日の場合は火曜定休）
  5. 浅草寺・駅からの徒歩ルート（多言語テキスト枠あり）
  6. Googleマップ口コミ（4.8pt・181件）＋口コミ投稿ボタン

---

### 6. ONLINE｜オンラインショップページ

- **目的：** WS体験後のEC再購入・遠方ファン向けの購買導線。体験→EC再購入の循環を設計する
- **主要コンテンツ：**
  1. EC商品一覧（カテゴリタブ絞り込み）
  2. 「体験した香りをもう一度」セクション（処方レシピQR連携）
  3. 破天荒を中央強調（看板商品）
  4. ギフトラッピング・熨斗対応案内
  5. LINE公式アカウント登録CTA（リピート促進）

### 7. RESERVE｜予約フォームページ（F2）

- **目的：** じゃらん依存脱却の核機能。顧客データ取得・LINE連携・リピート設計の起点
- **主要コンテンツ：**
  1. カレンダー型空き枠表示（全日表示・土日祝のみ開催）
  2. 時間帯選択（11:00〜12:00／13:00〜14:00／15:00〜16:00）
  3. プラン・人数選択（2名デフォルト：予約の82.2%が2名）
  4. 顧客情報入力フォーム（氏名・ふりがな・メール・電話）
  5. 予約完了・サンクスページ（LINE登録／口コミ誘導／アクセス確認）
  6. じゃらんリンク併記（移行期対応）

## 設計上の重要な注意事項

| 項目 | 内容 |
|---|---|
| WSへの誘導CTA設置ページ | WORKSHOPページ・RESERVEページのみ |
| WSへの誘導CTA非設置ページ | WANOWA各ページ・シリーズトップページ（WANOWAの香りはWSの調香材料と別物のため） |
| WSで使用する調香材料 | 12種のオードトワレ（トップ：ベルガモット・レモン・グレープフルーツ / ミドル：ラベンダー・ミュゲ・ダマスクローズ・アッサムティー・カシス・マグノリア / ラスト：ムスク・アンバー・サンダルウッド） |
| 国造ゆず産地 | 石川県 能美市 国造地区（有機栽培30年以上）※高知県は誤り |
| 全商品仕様 | 日本製（Japan Made）。香り成分は日本産を含むが全てではない |
| 破天荒・PERFUMERSの個別商品ページ | 現行inimu.jpにリンク（新HPでは制作しない） |
