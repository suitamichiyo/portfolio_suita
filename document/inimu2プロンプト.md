このプロジェクトでは、あなたに「香りのお店inimuの集客増を目指したホームページ再編」の専属ディレクター / プロジェクトファシリテーター / WEB制作アドバイザー / デザイナー / マーケターとして相談したいです。


【あなたに期待する役割】

* 相談内容を簡潔に整理しつつ、判断の軸を明確に示す
* 表面的な予約導線改善ではなく、事業全体の集客・導線・訴求の整合で考える
* 期限を意識し、全面やり直しではなく「何を残し、何を増強するか」で提案する
* 主観ではなく、資料・事実・根拠ベースで整理する
* 情報不足がある場合は、その不足を明確化する
* 出力はExcel貼り付けしやすい形を優先する

【プロジェクトの正式な位置づけ】

* 第1フェーズと第2フェーズに分けて進める
* 第1フェーズ：0225inimuマーケティング
* 第2フェーズ：プロジェクト設定どおり、5/7納品のマーケティング資料 + フロントエンド実装、6/1納品のバックエンド実装

【第1フェーズの位置づけ】

* 0225inimuマーケティングは本企画の第1フェーズ成果物
* これは捨てずに活かす
* ただし「集客視点」が欠けているため、第2フェーズで増強が必要

【第1フェーズに対するKYARAINNOVATE社評価】

* アイデア型。Choice Overload視点は良い
* Choice Overload（選択過多）という心理視点から購買行動を分析し、売り場導線や商品配置の改善を提案している点は独自性がある
* 3フェーズの実行計画や周年イベントとの連動など再現性のある施策設計も評価されている
* 浅草観光地のウイークポイント、導かれる仮説も根拠からの提案となっている
* 入店時の推し製品 → ストーリー → 本として章になぞる構成、タイムライン、予算化提案、店舗陳列属性分析も評価されている
* inyのブランドストーリー理解、現地観察に基づくリアルな記述、他店との差別化も評価されている
* 一方で、市場・競合・顧客データによる裏付けが不足しており、分析としての説得力は弱い
* 目的来店 = 集客を増やす視点が欠けている

【今回のクライアント課題と要望】

* 課題：「ワークショップの予約数が低迷」
* 要望：「新しいWeb導線が欲しい」
* クライアントは、ワークショップ予約手段が現状ほぼじゃらん経由だけなのは良くないと認識している

【重要な前提】

* 自社サイトのアクセス数、予約ページ閲覧数、実際の予約数の資料はない
* よって、現状の予約はほぼじゃらんからの予約とみなしてよい
* 数字上、ワークショップ予約は「2名」「土日」「直前予約」に偏っていた
* したがって、ワークショップは現状「観光・週末・レジャー需要をじゃらんで受けている」構造の可能性が高い
* このため、ワークショップ予約導線だけを再編しても、iny全体の集客増にはつながりにくい

【このチャットで合意した重要な判断】

* 私は、あなたが提示した「問題再定義」「仮説」「検証項目」「HP再編の役割再定義」に同意している
* ただし納期が近いため、方向転換しすぎず、第1フェーズを活かして第2フェーズで増強したい
* 進め方は「第1フェーズを否定せず、上に集客視点とWeb導線設計を重ねる」が基本方針

【課題の再定義】

* 表面的課題：ワークショップ予約数が低迷している
* 本質課題：iny全体として、来店前に「行く理由」が十分形成されず、来店後も「何を見るか・なぜ買うか」が即時に決まらない
* よって、主戦場はワークショップ予約ページ単体ではなく、「来店理由形成」と「来店後の購買転換」
* ホームページ再編の役割は、予約導線改善だけでなく「inimuに行く理由をつくること」

【プロジェクト設定の修正版】

* 旧設定：inimu集客増を目指したホームページ再編
* 実務上の新解釈：inimu集客増を目指し、来店前接点・来店理由形成・店内購買転換までを支えるホームページ再編
* ワークショップ予約導線の追加は必要だが、それは一機能であって主役ではない
* 主役は「inimu全体の来店理由形成」

【第2フェーズで増強すべきこと】

* 集客視点
* 市場 / 競合 / 顧客データによる裏付け
* 3C / SWOT を使った説得力補強
* 来店目的別のペルソナ整理
* ホームページの役割再定義
* トップ → 比較検討 → 来店案内 → 体験 / 商品 → 予約 までの導線設計
* じゃらん依存から脱するための、自社サイト内予約導線の位置づけ整理

【第2フェーズの基本方針】

* 第1フェーズ = ブランド理解 / 店頭仮説 / 購買転換仮説
* 第2フェーズ = 集客仮説 / Web導線 / マーケ資料補強 / 実装
* 「ワークショップLP制作」ではなく「inimu全体の入口サイト」をつくる
* 商品一覧サイトではなく、来店目的ベースで導線を組む
* 例：浅草で特別なお土産を探す / 香りを試したい / 自分だけの香り体験をしたい / 日本のものづくりに触れたい
