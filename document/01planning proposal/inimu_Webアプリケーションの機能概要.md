# inimu Webアプリケーション 機能概要
**サービス名：** inimu集客増を目指した来店理由形成型ホームページ再編  
**運営：** 株式会社 KARA INNOVATE  
**最終更新：** 2025年5月

---

## 技術スタック

| 技術カテゴリ | 使用技術 | 用途 |
|---|---|---|
| フロントエンド | HTML / CSS / JavaScript | Webサイト構築 |
| CSSフレームワーク | 未定（Tailwind CSS 等） | スタイリング効率化 |
| バックエンド | Java / Spring Boot | Webアプリケーション・管理システム |
| テンプレートエンジン | Thymeleaf | 管理システム画面表示 |
| O/Rマッピング | MyBatis | データベース操作 |
| データベース | MySQL（本番）/ H2（開発・テスト） | データ永続化 |
| 認証・セキュリティ | Spring Security | ログイン認証・権限管理（管理者／スタッフ） |
| APIサーバー | Spring Boot REST API | 予約空き枠取得・フォーム送信等の非同期通信 |
| デザインツール | Figma | ワイヤーフレーム・モックアップ |
| バージョン管理 | Git / GitHub | ソースコード管理・公開 |
| 開発環境 | Visual Studio Code | コーディング |
| AI活用 | Claude（claude.ai / Claude Code） | プロンプトプログラミング・実装支援 |
| デプロイ | Render | 本番環境への公開 |

## 機能一覧

### 認証・セキュリティ

| 機能名 | 概要 | 優先度 |
|---|---|---|
| ログイン認証 | メールアドレス＋パスワードによるログイン。Spring Securityで実装。未認証アクセスは /admin/login へリダイレクト | Must |
| 権限管理（ロールベース） | 管理者（全機能）とスタッフ（予約・顧客・問い合わせのみ）の2権限。売上レポート・設定画面は管理者のみアクセス可 | Must |

---

### 予約管理

| 機能名 | 概要 | 優先度 |
|---|---|---|
| 予約一覧・検索 | 日付・ステータス（確定／キャンセル／入場済）・氏名で検索・絞り込み。CSV出力対応 | Must |
| 予約詳細・編集 | 予約者情報・日時・人数・料金・ステータス変更・スタッフメモの確認と編集 | Must |
| 予約新規登録 | スタッフが管理画面から直接予約を登録。当日の飛び込み客対応用 | Must |
| 予約キャンセル・入場処理 | ステータスをワンクリックで「キャンセル済み」「入場済み」に更新 | Must |
| 予約カレンダー表示 | 月次・週次切り替え対応。土日祝の予約枠を色分け表示（空き・満員・要確認） | Must |
| 空き枠リアルタイム管理 | 時間帯別（11:00〜12:00／13:00〜14:00／15:00〜16:00）の残枠数をWebサイト側にAPI提供 | Must |
| 予約確認メール自動送信 | 予約完了時に顧客へ自動返信メール送信。メールテンプレートは管理画面から編集可能 | Must |

---

### 顧客管理

| 機能名 | 概要 | 優先度 |
|---|---|---|
| 顧客一覧・検索 | 氏名・メール・電話番号で検索。LINE登録済み・リピーター・初回フィルター対応。CSV出力可 | Must |
| 顧客詳細 | 基本情報・予約履歴・LINE登録ステータス・スタッフメモを一覧表示 | Must |
| 顧客新規登録 | スタッフが手動で顧客情報を登録できる | Should |

---

### 問い合わせ管理

| 機能名 | 概要 | 優先度 |
|---|---|---|
| 問い合わせ一覧・ステータス管理 | カテゴリ（一般／WS／ギフト・法人／OEM開発）とステータス（未対応／対応中／完了）で絞り込み | Must |
| 問い合わせ詳細・対応メモ | 問い合わせ内容の確認・対応メモ入力・ステータス変更 | Must |
| 自動返信メール送信 | 問い合わせ受付時に顧客へ自動返信。返信目安「3営業日（火〜日）以内」を明記 | Must |

---

### 商品管理

| 機能名 | 概要 | 優先度 |
|---|---|---|
| 商品一覧・表示管理 | カテゴリ別（WANOWA／破天荒／PERFUMERS等）表示。HP上の表示・非表示・看板商品フラグの切り替え | Should |
| 商品情報編集 | 商品名・説明文・価格・素材情報・産地・画像のアップロードと編集 | Should |

---

### 売上・レポート（管理者のみ）

| 機能名 | 概要 | 優先度 |
|---|---|---|
| 売上レポート | 期間指定（月次・年次・カスタム）でWS売上の集計・グラフ表示。CSV・PDF出力対応 | Should |
| WS実績サマリー | 予約件数・キャンセル率・曜日別分布・リードタイム別分布・じゃらん vs 自社予約比率の推移モニタリング | Should |

---

### 設定（管理者のみ）

| 機能名 | 概要 | 優先度 |
|---|---|---|
| 予約カレンダー設定 | 営業日（土日祝のみ）・時間帯・定員数・臨時休業日の登録と管理 | Must |
| WSプラン設定 | プラン名・料金（現在¥5,500/名）・所要時間・定員・公開/非公開の切り替え | Must |
| ユーザー管理 | スタッフアカウントの新規作成・編集・削除・権限設定（管理者／スタッフ） | Must |
| メールテンプレート設定 | 予約確認メール・問い合わせ自動返信メールのテンプレート編集 | Should |

---

### ダッシュボード

| 機能名 | 概要 | 優先度 |
|---|---|---|
| ダッシュボード（ホーム） | 本日の予約件数・今月売上速報・未対応問い合わせ件数・直近7日間の予約カレンダープレビューを一画面で確認 | Must |
