1 Controller メソッド一覧

タスク管理コントローラーの全メソッド
メソッド名HTTPURL引数戻り値処理概要
list GET /tasks Model model String タスク一覧を取得して Model に設定し、一覧画面を表示する。
show GET /tasks/{id} @PathVariable Long id, Model model String 指定IDのタスク詳細を取得して Model に設定し、詳細画面を表示する。
createForm GET /tasks/new Model model String 空の TaskForm を Model に設定し、新規作成フォーム画面を表示する。
create POST /tasks @Valid TaskForm form, BindingResult result String 入力値をバリデーションし、正常ならタスクを登録して一覧画面へリダイレクトする。
editForm GET /tasks/{id}/edit @PathVariable Long id, Model model String 指定IDのタスクを取得して TaskForm に設定し、編集フォーム画面を表示する。
update POST /tasks/{id} @PathVariable Long id, @Valid TaskForm form, BindingResult result String 入力値をバリデーションし、正常なら指定IDのタスクを更新して一覧画面へリダイレクトする。
delete POST /tasks/{id}/delete @PathVariable Long id String 指定IDのタスクを削除し、一覧画面へリダイレクトする。

3 Repository メソッド一覧

TaskMapper 必須
タスクマッパーの全メソッド
メソッド名引数戻り値SQL種別備考
selectAll なし List<Task> SELECT @Select を想定。created_at の降順で一覧取得する。
selectById Long id Task SELECT @Select を想定。主キー id を条件に1件取得する。
selectByUserId Long userId List<Task> SELECT @Select を想定。user_id を条件に絞り込み、期限順または作成日時順で取得する。
insert Task task void INSERT @Insert を想定。@Options(useGeneratedKeys = true) で採番IDを取得する。
update Task task void UPDATE @Update を想定。updated_at も同時に更新する。
updateStatus Long id, String status void UPDATE @Update を想定。status のみ更新し、必要に応じて updated_at も更新する。
delete Long id void DELETE @Delete を想定。主キー id を条件に物理削除する。
search String keyword List<Task> SELECT @Select を想定。title, content に対して LIKE 検索を行う。
countByUserId Long userId int SELECT @Select を想定。user_id ごとの登録件数を集計する。

selectByEmail String email User SELECT @Select を想定。email を条件に認証対象ユーザーを取得する。

Bean Validationアノテーション一覧
アノテーション対象フィールド検証内容エラーメッセージ
@NotBlank title null、空文字、空白のみ入力を禁止する。タスクタイトルの必須チェックに使用する。 「タイトルは必須です」
@Size(max=100) title タイトルの文字数を100文字以内に制限する。 「タイトルは100文字以内で入力してください」
@Size(max=1000) content 内容欄の文字数を1000文字以内に制限する。 「内容は1000文字以内で入力してください」
@NotNull dueDate 日付未選択を禁止し、期限入力を必須にする。 「期限を選択してください」
@Email email メールアドレス形式かどうかを検証する。ユーザー登録やプロフィール更新で使用する。 「正しいメールアドレスを入力してください」
@Pattern password 正規表現でパスワードの桁数や使用可能文字を検証する。例: 8文字以上の条件に使用する。 「パスワードは8文字以上で入力してください」
